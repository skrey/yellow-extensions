## Instalacja

[Pobierz rozszerzenie](https://github.com/datenstrom/yellow-extensions/raw/master/zip/polish.zip) i skopiuj plik zip do folderu `system/extensions`. Kliknij prawym przyciskiem myszy, jeśli korzystasz z Safari.
